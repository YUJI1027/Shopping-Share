# My Vue Shopping App

このプロジェクトは、Vue.jsを使用して構築されたシンプルなショッピングアプリです。ユーザーはアイテムを追加、削除、編集することができます。

## 機能

- アイテムの追加
- アイテムの削除
- アイテムの編集
- 残りのアイテム数の表示
- スマホ対応のレスポンシブデザイン

## セットアップ

1. リポジトリをクローンします。

   ```bash
   git clone <repository-url>
   ```

2. ディレクトリに移動します。

   ```bash
   cd my-vue-shopping-app
   ```

3. 依存関係をインストールします。

   ```bash
   npm install
   ```

4. 開発サーバーを起動します。

   ```bash
   npm run dev
   ```

5. ブラウザで `http://localhost:3000` にアクセスします。

## 使用技術

- Vue.js
- TypeScript
- Vite
- CSS (レスポンシブデザイン)

## スタイル

アプリはブルーを基調としたデザインで、スマホ対応のスタイルが適用されています。
